createOpportunityDocument

  • createOpportunityDocument(fileName, fileType, opportunityId): Promise<DocumentEntity>

  • Inserta en la base de datos los datos de un documento adjunto a una oportunidad.

    Esta función recibe un nombre de archivo y tipo de archivo, verifica si el documento ya existe en la base de datos para la oportunidad específica. Si el documento no existe, lo guarda en la base de datos, si no, lo actualiza.

    Parameters

    • fileName: string

      El nombre del archivo a ser guardado.

    • fileType: string

      El tipo MIME del archivo a ser guardado.

    • opportunityId: string

      El ID de la oportunidad a la cual se asociará el documento.

    Returns Promise<DocumentEntity>

    Una instancia de DocumentEntity que representa el documento guardado.

        message: string;

  • Elimina un registro de documento asociado a una oportunidad en la base de datos.

    Este método busca el documento especificado. Si el documento existe, lo elimina de la base de datos. Si no existe, lanza un error.

    Parameters

    • opportunityId: string

      El ID de la oportunidad asociada al documento a eliminar.

    • fileName: string

      El nombre del archivo que representa el documento a eliminar.

    Returns Promise<{
        message: string;
    }>

    Un objeto con un mensaje de éxito si el documento se elimina correctamente.

    Throws

    AppError si el documento no existe o si ocurrió un error al intentar eliminarlo.

getDocumentsByOpportunity

  • getDocumentsByOpportunity(opportunityId): Promise<DocumentEntity[]>

  • Recupera todos los registros de documentos asociados a una oportunidad específica.

    Este método consulta los documentos vinculados a la oportunidad indicada por su ID y devuelve una lista de documentos asociados a esa oportunidad. No devuelve los documentos en sí si no los registros de la base de datos.

    Parameters

    • opportunityId: string

      El ID de la oportunidad de la cual se quieren recuperar los documentos.

    Returns Promise<DocumentEntity[]>

    Una promesa de un arreglo de instancias de DocumentEntity, que representan los documentos asociados a la oportunidad.
