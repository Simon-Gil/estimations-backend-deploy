Class AWSFileStorageService

Servicio para la gestión de archivos usando AWS S3. Implementa la interfaz IFileStorageProvider y proporciona métodos para generar URLs firmadas para la subida, descarga, eliminación y verificación de existencia de archivos en un bucket S3.

checkExistingFile

  • checkExistingFile(opportunityId, fileName, fileType): Promise<boolean>

  • Comprueba si un archivo existe en el servicio de almacenamiento de S3.

    Utiliza el servicio AWS S3 para verificar si el archivo existe en el bucket especificado. Si el archivo no existe, se devuelve false. En caso de error, se lanza una excepción.

    Parameters

    • opportunityId: string

      El identificador único de la oportunidad asociada al archivo.

    • fileName: string

      El nombre del archivo a verificar.

    • fileType: string

      El tipo de archivo (por ejemplo, 'image/png', 'application/pdf').

    Returns Promise<boolean>

    Una promesa que se resuelve con true si el archivo existe, o false si no.
