Class ExcelUtility

Clase que proporciona funcionalidades para generar un documento Excel basado en un detalle de estimación de tareas. Esta clase permite la creación de hojas de trabajo con información estructurada sobre tareas, categorías, subcategorías, perfiles, y totales de estimaciones.
