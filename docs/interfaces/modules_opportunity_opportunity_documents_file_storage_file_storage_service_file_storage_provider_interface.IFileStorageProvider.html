Interface IFileStorageProvider

Interfaz para un proveedor de almacenamiento de archivos.

Define los métodos necesarios para interactuar con un servicio de almacenamiento de archivos, incluyendo la generación de URLs firmadas para subir, descargar, eliminar y comprobar la existencia de archivos.

interface IFileStorageProvider {
    checkExistingFile(opportunityId: string, fileName: string, fileType: string): Promise<boolean>;
    generateDeleteUrl(opportunityId: string, fileName: string, fileType: string): Promise<string>;
    generateDownloadUrl(opportunityId: string, fileName: string): Promise<string>;
    generateUploadUrl(opportunityId: string, fileName: string, fileType: string): Promise<string>;
}

checkExistingFile

  • checkExistingFile(opportunityId, fileName, fileType): Promise<boolean>

  • Comprueba si un archivo existe en el proveedor de almacenamiento.

    La función consulta el proveedor de almacenamiento para verificar si el archivo especificado ya está presente en el sistema. Retorna true si el archivo existe, false en caso contrario.

    Parameters

    • opportunityId: string

      El identificador único de la oportunidad asociada al archivo.

    • fileName: string

      El nombre del archivo a comprobar.

    • fileType: string

      El tipo de archivo (por ejemplo, 'image/png', 'application/pdf').

    Returns Promise<boolean>

    Una promesa que se resuelve con true si el archivo existe, o false si no.

generateDeleteUrl

  • generateDeleteUrl(opportunityId, fileName, fileType): Promise<string>

  • Genera una URL firmada para eliminar un archivo.

    La URL firmada generada permite que un archivo sea eliminado de un servicio de almacenamiento sin necesidad de autenticar la solicitud cada vez. La URL tiene un tiempo de expiración limitado.

    Returns Promise<string>
